# Informations de Financement des Dépendances - Production

Ce document référence les informations de financement/sponsoring des packages npm utilisés dans le projet Portail HARP en production.

**Date de génération** : Production  
**Commande utilisée** : `npm fund`

## Vue d'ensemble

Le projet utilise de nombreux packages open-source qui acceptent le financement/sponsoring pour maintenir leurs projets. Cette liste peut être utilisée pour :
- Comprendre l'écosystème des dépendances
- Identifier les opportunités de contribution/sponsoring
- Référencer les versions en production

## Notes

- Cette liste est générée depuis l'environnement de production
- Les versions peuvent différer entre développement et production
- Pour mettre à jour cette liste, exécuter `npm fund` en production et mettre à jour ce document
- Certains packages peuvent avoir plusieurs sponsors (affichés en arborescence)

## Commandes utiles

```bash
# Afficher les informations de financement
npm fund

# Afficher uniquement les packages directs
npm fund --depth=0

# Afficher les informations pour un package spécifique
npm fund <package-name>
```
